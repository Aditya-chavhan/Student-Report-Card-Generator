# 🎓 Student Report Card Generator (Java)

A simple **Student Report Card Generator** built using **Java** that calculates total marks, percentage, and grade based on user input.

This project demonstrates the use of **Object-Oriented Programming (OOP)** and basic data processing.

---

## 🚀 Features

* 🧑 Enter student details
* 📝 Input marks for 5 subjects
* ➕ Automatic total calculation
* 📊 Percentage calculation
* 🏅 Grade assignment based on performance
* 📄 Display formatted report card

---

## 🛠️ Tech Stack

* **Java**
* **OOP Concepts (Classes & Objects)**
* **Arrays**
* **Scanner Class (User Input)**

## ⚙️ How It Works

* User enters:

  * Student name
  * Marks for 5 subjects

* Program calculates:

  * Total marks
  * Percentage
  * Grade (A–F system)

* Results are displayed as a **report card**
